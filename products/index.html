P3tray.OS is a Windowing Enviornment (Eventually its own fully-fledged Linux Distribution!) on Debian based to work around the functionality of only the keyboard. This makes using programs and browsing the internet significantly faster if you hate having to reach for your mouse, and especially if you get board of the blinking white cursor at the terminal. It infuses the terminal functionality and gives it a pleasing graphical output, best of both worlds!

Please note that P3tray.OS is undergoing a second re-write due to its code having major performace updates as well as increasing its maintainability in the long run. The main reason for this is my significant improvments in the Python 3 programming language over the years. Its current and previous alpha and beta distributions source code is currently unavailable. Pre-Alpha distributions are available on Repl.it.
